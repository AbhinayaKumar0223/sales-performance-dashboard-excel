# 📊 Sales Performance Dashboard (Microsoft Excel)

## 📌 Project Overview

This project is an interactive Sales Performance Dashboard built in Microsoft Excel to analyze business sales data and present meaningful insights through dynamic visualizations.

The dashboard enables users to monitor sales performance using KPI cards, PivotCharts, and interactive slicers.

## 📈 Dashboard Features

- 🛒 Category-wise Sales Analysis
- 🌍 Region-wise Sales Analysis
- 🏆 Top 10 Subcategories by Sales
